Репозиторий для одноимённого проэкта. Выложен релиз (версия 1.1). Каждый новый вариант правки - отдельный форк.
По завершении хотя бы одной правки та становится кандидатом на основу дальнейшей разработки.
Также для исторической справки и референсов предоставлены сценарные файлы демо-версии 2009 года с наработками по 3-4 дням, версии, выложенные Сорком (Sorc_ver) и Смолевым (1.0_ver).
Различия в скрипте между ыерсиями 1.0 и 1.1 представлены в папке diff файлом diff.txt; более компактная версия находится там же и разбита по файлам в соответствии с версией 1.0.
