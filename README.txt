Репозиторий для одноимённого проэкта. На данный момент лишь основные сценарные файлы версии, выложенной Смолевым. Каждый новый вариант правки - отдельный форк.
По завершении хотя бы одной правки та становится кандидатом на основу дальнейшей разработки.
Также для исторической справки и референсов предоставлены сценарные файлы демо-версии 2009 года с наработками по 3-4 дням и версия, выложенная Сорком.
